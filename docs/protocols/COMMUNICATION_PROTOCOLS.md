# Communication Protocols

**Version:** 1.0
**Source:** Derived from `docs/ORGANIZATIONAL_STRUCTURE.md` and `docs/OPERATIONS_MANUAL.md`.

## 1. Daily Standup (Async)
**Frequency:** Every Morning
**Format:**
-   **Completed Yesterday:** Specific deliverables.
-   **Today's Priorities:** 1-3 specific tasks.
-   **Blockers:** Dependencies or risks.
-   **Insights:** Learnings or questions.

## 2. Weekly Engineering Sync
**Time:** Friday 2 PM MST
**Attendees:** All 6 Sub-Agents + Executive Engineer
**Agenda:**
1.  Progress Review (15 min)
2.  Cross-Team Dependencies (15 min)
3.  Security & Quality (10 min) - Claude Code findings
4.  Strategic Decisions (15 min)
5.  Next Week Priorities (5 min)

## 3. Monthly Leadership Sync
**Time:** First Friday of Month, 10 AM MST
**Attendees:** Founder + Manus Agents
**Agenda:**
-   Phase Milestones Review
-   Strategic Direction
-   Team Health

## 4. Escalation Paths
**Immediate (Same Day):**
-   Security Vulnerability
-   Values Misalignment
-   Major Technical Blocker (>2 days)
-   **Action:** Flag in task with "ESCALATION:" prefix. notify Founder.

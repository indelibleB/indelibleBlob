# Solana Documentation

Solana is the high-performance blockchain designed for mass adoption.

## Core Concepts
- **Accounts**: The fundamental building block of Solana's state.
- **Fees**: Deterministic transaction costs.
- **Transactions**: Atomic execution of instructions.
- **Programs**: Stateless code that processes instructions (smart contracts).
- **PDA (Program Derived Address)**: Enables programs to control accounts.
- **CPI (Cross Program Invocation)**: Composability between programs.

## Client Usage
Solana provides robust SDKs for interacting with the blockchain, essential for building dApps that will be featured in the Seeker dApp store.

# Walrus

A decentralized blob store using Sui for coordination and governance.

## Documentation
Our main documentation is available [as a Walrus Site](https://docs.wal.app) (see [the documentation](https://docs.wal.app/walrus-sites/intro.html) for further information on what this means) and on [GitHub Pages](https://mystenlabs.github.io/walrus).

## Repository structure
- [contracts](https://github.com/MystenLabs/walrus/blob/main/contracts) contains all smart contracts used by Walrus for coordination and governance.
- [crates](https://github.com/MystenLabs/walrus/blob/main/crates) contains all Rust crates related to Walrus including binaries for storage nodes and clients.
- [docker](https://github.com/MystenLabs/walrus/blob/main/docker) contains Dockerfiles and docker-compose setups for building and running Walrus.
- [docs](https://github.com/MystenLabs/walrus/blob/main/docs) contains high-level technical and design documentation about Walrus.
- [scripts](https://github.com/MystenLabs/walrus/blob/main/scripts) contains tools used for evaluating and testing the code.
- [setup](https://github.com/MystenLabs/walrus/blob/main/setup) contains setup scripts and configuration files.

## Using the Walrus client
If you just want to use the Walrus Mainnet or Testnet, please follow the [setup instructions in our documentation](https://docs.wal.app/usage/setup.html).

# Solana Mobile Stack (SMS) & Seeker Documentation

Solana Mobile is a mobile development stack for building Web3-native mobile dApps.

## Core Components
### Seeker Device
The "Definitive Web3 Mobile Device" featuring:
- **Seed Vault**: Hardware-backed key custody.
- **Seeker ID**: On-chain identity integration.
- **SKR Token**: Native asset of the Solana Mobile economy.

### TEEPIN (Trusted Execution Environment Platform Infrastructure Network)
A critical feature for Indelible Blob's "Proof of Origin":
- **Verifiable State**: Devices attest to identity, boot status, and software authenticity.
- **Hardware Backed Trust**: Trusted Execution Environment (TEE) delivers tamper-resistant proofs of state for on-chain verification.

### Solana dApp Store
- **Gatekeeper Free**: Direct distribution to 150,000+ users.
- **0% Platform Fees**: Developers keep 100% of revenue.
- **PWA Support**: Support for converting Progressive Web Apps (like Indelible Blob) to APKs for store listing.

## Developer Resources
- **Mobile Wallet Adapter (MWA)**: Protocol for connecting mobile dApps to wallet apps.
- **React Native / Kotlin / Flutter SDKs**: Native development tools.

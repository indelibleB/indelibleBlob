# Solana Mobile Developer Documentation

## Overview

Solana Mobile Stack (SMS) provides a comprehensive toolkit for building mobile dApps on Android.

## React Native

Build mobile apps with JavaScript and reuse familiar Solana web libraries.

- **Mobile Wallet Adapter (MWA)**: Connects your dApp to compliant wallet apps on the device, enabling signing and sending transactions.
- **Solana Web3.js**: Standard library for interacting with the Solana blockchain.

## dApp Store

The Solana dApp Store is an app distribution channel on Saga and Seeker devices.

- **Submitting Apps**: Requires adherence to dApp policies.
- **APK Distribution**: Allows direct updates without centralized gatekeepers.

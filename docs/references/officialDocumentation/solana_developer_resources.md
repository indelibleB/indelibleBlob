# Solana Developer Resources

A manual for joining the Solana ecosystem. By builders for builders.

## Learning Paths
- **Solana Development Course**: Comprehensive guide to building on Solana.
- **Solana Bootcamp**: Intensive training for new developers.
- **Solana Bytes**: Short, focused educational content.
- **Build on Solana by Rise In**: Structured learning for dApp development.

## Tutorials & Guides
- **Optimize Compute Usage**: Best practices for high-performance programs.
- **Get Devnet SOL**: Instructions for faucets and airdrops.
- **Solana Docs**: Official technical documentation.
- **Anchor Docs**: Framework for building secure Solana programs.

## SDKs
- **Official**: `@solana/web3.js`, `@solana/react-hooks` (Built by Anza).
- **Community**: `solders`, `solana-py`, `solana-go`.

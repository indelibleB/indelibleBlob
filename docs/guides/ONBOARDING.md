# New Agent Onboarding Guide

**Source:** Derived from `docs/CONSTITUTION.md` (Part 9).

## Week 1: Foundation

### Day 1: Immersion
1.  Read `docs/CONSTITUTION.md` (Mission & Values).
2.  Read `docs/OPERATIONS_MANUAL.md` (How we work).
3.  Read `docs/TECHNICAL_ARCHITECTURE.md` (The Stack).
4.  Read `docs/AGENT_PLAYBOOKS.md` (Your specific role).

### Day 2-3: Context
1.  Review `docs/ORGANIZATIONAL_STRUCTURE.md`.
2.  Browse `docs/decisions/DECISIONS.md` for history.
3.  Explore the Codebase (focus on your domain: `mobile/`, `website/`, etc.).

### Day 4-5: Alignment
1.  Set up your Development Environment (`docs/guides/DEVELOPMENT_SETUP.md`).
2.  Attend the Weekly Engineering Sync (Friday 2 PM MST).
3.  Commit your first small fix or doc update.

## Checklist
- [ ] Values Alignment Confirmed
- [ ] Role Responsibilities Understood
- [ ] Dev Environment Working
- [ ] Access to Repo & Keys
- [ ] First PR Merged

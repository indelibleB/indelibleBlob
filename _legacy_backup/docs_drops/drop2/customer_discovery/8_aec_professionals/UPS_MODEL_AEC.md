# UPS Model: AEC Professionals Segment
## User-Problem-Solution Framework for indelible.Blob

**Target Segment:** Architects, Engineers, Construction Managers  
**Market Size:** $10T+ global construction industry  
**Phase:** Phase 2 (Months 25-36) - Requires 3D spatial data capture  
**Pricing:** $29/month (Professional), $10k-$100k/year (Firm/Project)

---

## USER: Who Has the Problem?

### Persona 1: Construction Manager / Site Supervisor
- Manages construction projects ($1M-$100M+)
- Documents site progress, safety, quality
- Needs verified documentation for disputes
- $70k-$150k/year salary
- Liability for project delays, defects, safety incidents

**Pain:** Construction disputes cost $50k-$5M+, need verified documentation, photos questioned in litigation

---

### Persona 2: Structural Engineer / Civil Engineer
- Designs buildings, bridges, infrastructure
- Inspects sites, documents conditions
- Needs verified as-built documentation
- $65k-$130k/year salary
- Professional liability for design errors

**Pain:** Liability claims, need proof of site conditions, documentation disputes

---

### Persona 3: Architect / BIM Manager
- Designs buildings, manages BIM models
- Needs verified spatial data for as-built models
- Coordinates with contractors, engineers
- $60k-$120k/year salary
- Responsible for design-build coordination

**Pain:** As-built vs. design discrepancies, need verified spatial scans, coordination errors cost millions

---

## PROBLEM: What Painful Problem Do They Experience?

### Core Problem

**Construction projects face $50k-$5M+ disputes over site conditions, progress, and quality. Documentation is questioned in litigation, costing millions in expert witnesses and delays. AEC professionals need verified spatial documentation to prove site conditions, track progress, and defend against liability claims. Current tools don't provide cryptographic proof.**

---

### Problem Layers

**Layer 1: Construction Disputes**
- 90% of construction projects face disputes
- Documentation questioned: "Was this photo taken at this site? When?"
- Disputes cost $50k-$5M+ per project
- **Impact:** Financial losses, project delays, reputation damage

**Layer 2: Liability & Professional Insurance**
- Engineers/architects face liability claims
- Need proof of site conditions at time of design
- Professional insurance requires documentation
- **Impact:** Claims cost $100k-$10M+, insurance premiums increase

**Layer 3: As-Built Documentation**
- Need verified 3D spatial data for as-built models
- Discrepancies between design and as-built cost millions
- No way to verify spatial scan authenticity
- **Impact:** Coordination errors, rework costs, project delays

**Layer 4: Progress Tracking & Payment**
- Owners need verified progress documentation for payment
- Contractors dispute progress claims
- Photos can be manipulated or misdated
- **Impact:** Payment delays, cash flow problems, disputes

---

## SOLUTION: How Does indelible.Blob Solve the Problem?

### Core Solution

**indelible.Blob provides verified spatial documentation for construction projects. GPS + blockchain prove when/where photos and 3D scans were captured. Walrus storage ensures immutability. This creates court-admissible evidence for disputes, verified as-built documentation, and automated progress tracking.**

---

### Solution Components

**1. Verified Site Documentation**
- GPS proves photo/scan location (correct site)
- Timestamp proves when (project phase)
- Blockchain creates immutable record
- **Benefit:** Court-admissible evidence, disputes resolved

**2. 3D Spatial Scanning (Phase 2)**
- LiDAR integration for as-built documentation
- Verified spatial accuracy (GPS + orientation)
- BIM integration (Revit, Navisworks)
- **Benefit:** Accurate as-built models, coordination errors prevented

**3. Automated Progress Tracking**
- Verified photos/scans at milestones
- Smart contracts trigger payments based on verified progress
- Owner verification interface
- **Benefit:** Faster payments, reduced disputes

**4. Liability Protection**
- Verified documentation of site conditions
- Chain of custody for inspections
- Expert witness cost reduction ($5k-$20k → $0)
- **Benefit:** Liability claims defended, insurance costs reduced

---

### Competitive Differentiation

**vs. Construction Management Software (Procore, PlanGrid):** Verification + immutability vs. just documentation  
**vs. 3D Scanning Services:** Verified provenance vs. unknown origin  
**vs. Expert Witnesses:** $29/month vs. $5k-$20k per dispute  
**vs. Manual Documentation:** Automated + cryptographic proof vs. error-prone
